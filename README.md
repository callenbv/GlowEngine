# GlowEngine
Custom C++ 3D game engine built on DirectX 11 featuring an ECS architecture, 
real-time rendering pipeline, and physics system.

## Demo
When run, the engine loads a simple scene demonstrating:

- Entity Component System (ECS)
- Real-time mesh rendering with DirectX 11
- BoxCollider physics and collision resolution
- Camera controls and player movement (WASD + mouse)

Helpful Controls:
- WASD: Move camera
- Space: Move up
- Shift: Move down
- Mouse: Look
- Right-Click: Pivot in-scene
- TAB: Toggle fullscreen
- G: Toggle Debug-Visuals
